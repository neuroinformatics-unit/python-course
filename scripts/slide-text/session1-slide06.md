# What is an Interactive Development Environment (IDE)

- Most (all?) programming languages do not require an IDE
- You can simply write code into a text file and then feed that to the interpreter
- Writing code into a text file with notepad is a bad user experience
- An IDE will hold your hand while you write code
- Autocomplete – it will suggest the correct ways for you to finish a statement
- Code check – It will underline in red any mistakes you make before you even run the code. It will also explain those errors to you.
- Allows you to run the code from the editor itself
- Many many more features
