- Function return values
- def a_func(a, b):
- print(a + b)
- def b_func(a, b):
- return a + b
- a_func(10, 20)
- b_func(10, 20)
- result = b_func(10, 20)
- print(result)
- Functions can return one or more values
- The values to return are signified by the return keyword
- Some functions don’t return anything!
