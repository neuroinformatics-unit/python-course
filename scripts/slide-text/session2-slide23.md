# Other ways of creating collections

- Instead of using [] for lists we can use the keyword “list()” and provide a different type of collection
- Instead of () for tuples I can use the keyword “tuple()” and provide a different type of collection
- Instead of {} and key value pairs for dictionaries I can use the keyword “dict()” and provide a list that contains lists which each contain a key value pair
- Instead of {} for sets I can use the keyword “set()” and provide a different type of collection
- These are also a useful way of converting between types of collection
