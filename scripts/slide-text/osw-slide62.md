- Immutable (i.e. cannot be edited)
- When would you use a list vs tuple?

## Notes

Basic answer: when you know the collection will be of fixed length and will never need to be changed then a tuple will be faster than a list
More advanced answer: Tuples are faster, hashable (can be used as keys for dictionary), write-protect
